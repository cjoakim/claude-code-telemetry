# README for the python/ directory

## Create the virtual environment

Execute the sh script for macOS/linux, ps1 script for Windows 11.

```
./venv.sh
- or -
.\venv.ps1
```

## Activate the virtual environment

Execute the sh script for macOS/linux, ps1 script for Windows 11.

```
source .venv/bin/activate
- or -
.\.venv\Scripts\activate
```
